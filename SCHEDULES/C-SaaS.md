## 1. SERVICE DESCRIPTION
Cloud-based software provided on subscription basis.

## 2. SUBSCRIPTION MODELS
### User-Based
- Per user per month
- Annual commitment: 20% discount

### Feature-Based
- Tiered feature sets
- Custom packages available

## 3. PRICING TIERS
### Basic
- $29/user/month
- Core features only

### Professional
- $99/user/month
- Advanced features + API access

### Enterprise
- Custom pricing
- All features + custom development

## 4. SERVICE COMMITMENTS
### Uptime
- 99.9% uptime SLA
- Credit for downtime > 0.1%

### Support
- Business hours: Basic/Professional

- Phone support: Enterprise only

### Encryption
- Data encrypted at rest and in transit
- TLS 1.2+ required

### Backup
- Daily automatic backups
- 30-day retention standard
- Extended retention available

- Maintain account security
- Comply with acceptable use policy
- Pay invoices on time

## 7. TERMINATION
### By Customer

- Annual: At contract end

### By NOVATOR

- 30-day cure period
